# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.4] - 2026-05-13

### Added
- Contribution heatmap visualization for learning consistency tracking
- Weekly calendar view to track daily goal completions
- Theme picker for sharing insights
- Pomodoro timer with custom duration support
- AI-powered goal generation using Google Generative AI (Gemini)
- Firebase Cloud Messaging for streak reminders
- Dark mode and system theme support

### Features
- Track-based learning system for organizing goals
- Reflection-first completion requiring learning insights
- Streak system with daily notifications
- Local-only data storage with Hive database
- Privacy-first approach with no analytics

## [1.0.0] - 2026-01-01

### Added
- Initial release
- Basic goal tracking
- Streak counter
- Local data persistence
- Flutter cross-platform support
